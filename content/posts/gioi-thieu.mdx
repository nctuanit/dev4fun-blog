# Chào Mừng Đến Với Dev4Fun Blog!

Xin chào các bạn! Rất vui được giới thiệu đến các bạn **Dev4Fun Blog** - một nơi để chia sẻ đam mê lập trình và công nghệ.

## Dev4Fun Blog Là Gì?

Dev4Fun Blog được xây dựng với mục tiêu tạo ra một không gian để:

- **Chia sẻ kiến thức** về các công nghệ mới nhất
- **Hướng dẫn từng bước** các dự án thực tế
- **Tips & Tricks** giúp code hiệu quả hơn
- **Cộng đồng** developer Việt Nam kết nối và học hỏi

## Công Nghệ Sử Dụng

Blog này được xây dựng bằng các công nghệ hiện đại:

| Công nghệ | Mô tả |
|-----------|-------|
| Next.js 15 | Framework React với App Router |
| Tailwind CSS 4 | Utility-first CSS framework |
| MDX | Markdown + JSX cho nội dung |
| TypeScript | Type-safe JavaScript |
| Bun | JavaScript runtime & package manager |

### Tại Sao Chọn Next.js?

Next.js mang lại nhiều lợi ích:

1. **App Router** - Routing hiện đại và mạnh mẽ
2. **Server Components** - Tối ưu performance
3. **Static Generation** - Tốc độ load nhanh chóng
4. **SEO Friendly** - Tối ưu cho search engine
